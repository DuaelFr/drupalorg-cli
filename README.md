Drupal.org CLI
--------------

A command line tool for interfacing with Drupal.org. Uses the Drupal.org REST API.

## Commands

````

Available commands:
  help            Displays help for a command
  list            Lists commands
 drupalci
  drupalci:list   Lists test results for an issue
  drupalci:watch  Watches a Drupal CI job

````
